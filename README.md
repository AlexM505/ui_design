# UI design

### Aplicacion movil

El objetivo es implementar los conocimientos adquiridos en una sola app.
Tanto diseño de intefaces visuales como aquitecturas como MVVM.

Tecnologias y librerias utilizadas (y por utilizar):

1. Kotlin
2. AndroidX
3. Navigation
4. Konfetti
5. Ripplebackground
6. Lottie
7. Mapboxsdk
8. ElasticViews
9. Retrofit2
10. Coroutines
11. CircleImageView
